# 🌌 Yosia's VS Code "Antigravity" Configuration

Selamat datang di repositori konfigurasi Visual Studio Code khusus milik saya! Konfigurasi ini dirancang dengan tema **SynthWave84**, mengutamakan estetika "Neon/Cyberpunk", kenyamanan mata saat ngoding lama (Anti-Pusing), dan efisiensi produktivitas yang maksimal.

## ✨ Fitur Utama

- 🎨 **Visual Core & Glow**: Menggunakan tema **SynthWave84** yang dimodifikasi dengan aksen *neon glow* kustom pada *line highlight* dan *tab active*. UI dibuat imersif dan minim distraksi (tanpa minimap, scrollbar tersembunyi).
- ⌨️ **Tipografi Premium**: Menggunakan kombinasi font **JetBrains Mono** dan **FiraCode Nerd Font** dengan dukungan *ligatures*, jarak huruf (letter spacing) yang pas, dan baris yang lega.
- ⚡ **Auto-Format & Prettier**: Sudah diatur agar otomatis merapikan kode (*Format on Save*) untuk bahasa populer (JavaScript, TypeScript, React, HTML, CSS, JSON).
- 🎨 **Tailwind CSS Boost**: Autocomplete cerdas untuk Tailwind CSS dan Emmet.
- 🐘 **Laravel & PHP**: Pengaturan bawaan untuk Intelephense agar *intellisense* jalan lancar.
- 🐍 **Python & Java**: Support penuh Pylance dan Oracle Java formatter.
- 🚀 **Produktivitas Maksimal**:
  - **File Nesting**: File config (.env, package.json, tailwind.config.js, dll.) disembunyikan di bawah file utamanya agar *Explorer* tetap rapi.
  - **Sticky Scroll**: Mencegah tersesat di dalam file yang memiliki ribuan baris kode.
  - **Linked Editing & Emmet**: Ubah tag pembuka, tag penutup ikut berubah. Support untuk React dan Laravel Blade.
  - **Auto Organize Imports**: Membersihkan dan mengurutkan import otomatis setiap kali disave (sangat berguna untuk Next.js / React).

## 🚀 Cara Menggunakan

1. **Install Tema & Font**:
   - Install ekstensi **SynthWave '84** dari Marketplace VS Code.
   - Install ekstensi **Material Icon Theme**.
   - Pastikan font **JetBrains Mono** dan **FiraCode Nerd Font** sudah terinstall di sistem kamu.

2. **Terapkan Pengaturan**:
   - Buka VS Code.
   - Buka **Settings (JSON)** dengan cara: `Ctrl + Shift + P` -> ketik `Preferences: Open User Settings (JSON)`.
   - Copy seluruh isi file `setting.json` dari repositori ini dan paste ke dalam file pengaturan `settings.json` milikmu.

3. **Install Ekstensi Pendukung (Opsional namun direkomendasikan)**:
   - Prettier - Code formatter
   - Tailwind CSS IntelliSense
   - PHP Intelephense
   - Python & Pylance
   - Extension Pack for Java
   - Comment Translate

## 🖥️ Terminal & Shell

Terminal default telah disesuaikan menggunakan PowerShell dengan font pendukung untuk Oh My Posh (`FiraCode Nerd Font`). *Shell integration* bawaan dimatikan untuk memastikan tampilan *prompt* (seperti Oh My Posh) tidak rusak atau tumpang tindih.
